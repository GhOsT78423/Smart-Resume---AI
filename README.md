# SmartResume AI

SmartResume AI is an AI-powered Resume Builder developed using Python and Streamlit.

## Features

- Generate professional resumes
- AI-generated career objective
- Resume preview
- Export as PDF
- Export as DOCX

## Technologies

- Python
- Streamlit
- OpenAI API
- python-docx
- FPDF2

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
